针对怪物猎人世界的黑龙所作出的出招ai重构,

因为实在受不了黑畜一纳刀就出蓄力火和孕吐,所以1.0版本先把黑龙的蓄力爹和孕吐母飞了,让我们举手高呼:黑八分钱.

怪物的行为通过thk文件里揭示的决策树控制,各节点都有对应的招式,详情请前往https://www.bilibili.com/opus/932887546921222162?spm_id_from=333.1007.0.0

本项目尽量对黑畜的ai做最小的改动,力求修改后的每一招是符合原ai的(当然也挺难的,主要是我太菜了打太少了看不出来)

注:三阶段的蓄力火没有删除,因为是另一个决策树节点控制的,而且三阶段蓄力火是好招

感谢b站up主九世深渊和Eigeen本征开源

使用方法:

进入.\Monster Hunter World\nativePC

创建文件夹em(如果没有)

进入.\Monster Hunter World\nativePC\em

创建文件夹em013(如果没有)

进入.\Monster Hunter World\nativePC\em\em013

创建文件夹00(如果没有)

进入.\Monster Hunter World\nativePC\em\em013\00

创建文件夹data(如果没有)

进入.\Monster Hunter World\nativePC\em\em013\00\data

将下载的决策树文件解压到data里面
